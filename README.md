# Caelum Lab

## Caelum Lab é um site educacional criado para acompanhar o canal Caelum no YouTube.

### O objetivo do projeto é explicar conceitos de:

- Física
- Mecânica orbital
- Astronáutica
- Exploração espacial

utilizando exemplos práticos em jogos como Kerbal Space Program e Spaceflight Simulator.

Além do conteúdo do canal, o site também apresenta explicações teóricas e conceitos científicos aplicados às missões e construções mostradas nos vídeos.

---

# 🌐 Site

# 🎯 Objetivo do projeto

O objetivo do Caelum Lab é tornar conceitos de física e exploração espacial mais acessíveis, especialmente para pessoas que:

- gostam de jogos espaciais
- querem entender como foguetes e órbitas funcionam
- procuram conteúdo em português sobre esses temas
